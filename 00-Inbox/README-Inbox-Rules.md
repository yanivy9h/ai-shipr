# Inbox Rules

Purpose: capture raw inputs (Slack, Email, Miro, notes).

Rule:
- Inbox is temporary.
- Anything that matters gets converted into:
  - Hypothesis (H)
  - Initiative (I)
  - Proof (P)
  - Relationship note (R)
- Otherwise delete/archive.
