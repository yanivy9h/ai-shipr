# EXP-001 — Short-Form Video A/B Test

## Linked Hypothesis
- [[HYP-001]] — Short drill videos (under 2 min) increase daily completion rate by 40%

## Linked Initiative
- [[INI-001]] — Short-form drill videos for mobile

## Status
In Progress

## Method
A/B test — 50/50 split of new users (mobile only)
- **Treatment (50%)**: Short-form drill videos, all under 2 minutes. Ball Handling Foundation level was reformatted first (Crossover Basics, Between the Legs, Figure 8 Dribble) — these are the highest-traffic drills for new users.
- **Control (50%)**: Original long-form drill videos (average 8–12 minutes)

## Experiment Window
- Start date: 2026-02-24
- End date: 2026-03-10 (14 days)
- Early read: 2026-03-03 (day 7 — check for obvious signal in either direction)

## Who's in the Test
- New users only (first session after signup)
- Mobile only — excluded tablet and desktop sessions
- Existing users excluded to avoid habit contamination
- Minimum 300 users per arm required for statistical confidence — currently at ~240/arm as of 2026-03-01

## Primary Metric
- **Drill completion rate**: % of drill sessions that reach the "log completion" step
- **Baseline (control)**: 38%
- **Success threshold**: ≥53% in treatment arm (+40% lift)
- **Failure threshold**: <44% (+15% or less) — if below this, short-form is not the lever

## Secondary Metrics (directional only)
- Average session length — expect to drop from ~12 min to under 4 min in treatment
- Day-3 return rate — does a higher first-session completion rate drive return?
- Drill abandonment rate — where in the video do users drop off?

## Early Signal (Day 7 Read — 2026-03-03)
- Treatment completion rate: 51%
- Control completion rate: 38%
- Lift: +34% — directionally positive, below the 40% threshold but within noise at this sample size
- Average session length (treatment): 3.8 min vs. 11.4 min (control) — confirmed format change working
- No negative signal on drill quality perception (no increase in "too short" feedback)

## Risks Being Watched
- **Quality concern**: Players in treatment may feel underprepared — monitoring re-watch rate and skip rate
- **Novelty effect**: Early lift could be novelty, not real behaviour change — day-14 read will tell
- **Skew by drill type**: Ball Handling Foundation drills converted first; Shooting and Finishing still in original format for both arms

## Next Decision Point

- If day-14 reads +15–40%: Extend test by 7 days, check if Shooting and Finishing data changes the picture.
- If day-14 reads <+15%: Kill short-form as primary lever. Schedule retrospective. Escalate retention problem upstream.
