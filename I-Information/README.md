# Information

This folder holds everything AI-SHIPR reads from the outside world —
and everything you give the system to learn from.

Three types of content live here:

- **Resources/** — Raw inputs: articles, research, competitor notes, PM frameworks, anything you want the system to learn from or reference.
- **Templates/** — Reusable document and output templates: PRDs, specs, stakeholder updates, decision logs, anything you generate repeatedly.
- **Integrations/** — Data piped in from connected tools: Slack threads, email summaries, Miro boards, Linear exports. Each subfolder matches one tool.

The Learner Agent (when active) monitors Resources for new drops and surfaces insights.
Integrations are updated by their respective reader agents or manual exports.
