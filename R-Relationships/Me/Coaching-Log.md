# Coaching Log

A running record of situations brought to PM-Coach.

PM-Coach reads this file at the start of every session to understand patterns, avoid repeating advice, and build on past decisions.

---

## How This Works

- After each coaching session, PM-Coach offers to write an entry here for your review.
- You can also add entries manually after any significant interpersonal situation.
- When a pattern appears 3 or more times, PM-Coach will flag it and suggest adding it to PM-Profile.md.

---

## Entry Format

```
### [Date] — [Situation Type]

**Situation:** [1-2 sentence description of what you were facing]
**Mode:** Soft Skills / Stuck on Options / Post-Situation Debrief
**Approaches surfaced:** [brief list of what was considered]
**What you chose:** [direction taken, if known]
**Pattern:** [recurring theme or named tendency, if applicable]
```
