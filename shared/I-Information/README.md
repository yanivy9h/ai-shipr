# Shared Information

Org-wide templates, integrations, and resources that all PMs use.
This is the team layer of I-Information — things that belong to the organization, not to any individual PM.

Each PM still has their own `I-Information/` folder for personal resources and product-specific integrations.
Files here take precedence when there is a conflict with personal templates.

---

## What lives here

| Folder | What it contains |
|--------|-----------------|
| `Templates/` | Org-standard document formats — PRD template, stakeholder update format, OKR framing, 1:1 structure |
| `Integrations/` | Company-level tool configurations — org Linear workspace, shared Slack channels, company Figma project |
| `Resources/` | Org-wide research and market context — competitor analysis, market reports, company positioning docs |

---

## Ownership rule

The VP fills and maintains this folder.
PMs read from it — they do not edit it.
When a PM needs a company-level resource that isn't here, they flag it to the VP rather than creating their own version.
